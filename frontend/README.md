# Городской портал — фронтенд

Интерфейс в стиле макетов: главная, регистрация, вход, форма обращения, личный кабинет «Мои заявки», админ-панель с CRM и аналитикой.

## Запуск

```bash
cd frontend
npm install
npm run dev
```

Откройте http://localhost:5173. Бэкенд должен быть запущен на http://127.0.0.1:8000 (или укажите `VITE_API_URL` в `.env`).

## Сборка

```bash
npm run build
```

Статика будет в `dist/`. Для продакшена задайте переменную окружения `VITE_API_URL` с URL бэкенда.

## Роли

- Обычный пользователь: главная, регистрация/вход, подача обращения, личный кабинет (список заявок, поиск, фильтр по категориям).
- Админ (`role === 'admin'` в БД): дополнительно доступна страница «Админ-панель» с аналитикой (графики по категориям и статусам) и CRM (таблица обращений, смена статуса).

Чтобы выдать роль администратора, обновите в БД поле `role` у нужного пользователя на `admin`.
